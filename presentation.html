Bài học
Lessons Learned

What went well

  • Adapter pattern let us swap AI/storage/userstore/vector backends without touching business logic.
  • Nova Lite → Claude Sonnet 4.5 was a one-line tfvars change.
  • Terraform 8-module structure made deploy reproducible and teardown one-command.
  • Presigned URL upload solved the 6MB Lambda payload limit cleanly.

What we'd do differently

  • Test Bedrock model access and Lambda concurrency limits on Day 0.
  • We lost hours debugging 503s caused by a 10-concurrent-execution account limit.
  • Remove redundant fallback search earlier; S3 keyword scanning caused 90s timeouts under load.

Failure mitigated

  • Boto3 read_timeout=10s was too aggressive for Bedrock RetrieveAndGenerate.
  • Three fallback model attempts pushed total latency beyond API Gateway's 30s limit.
  • Fix: timeout raised to 25s and expensive S3-scanning fallback removed.

Khanmigo engineer would ask

  • "How do you isolate notes from different subjects for the same student?"
  • Current scope filters by user_id only.
  • Production fix: add course_id metadata at KB ingestion time and filter retrieval per subject.
